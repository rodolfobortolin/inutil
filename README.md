# A CLASSE INÚTIL

**Como a Inteligência Artificial Vai Tornar Bilhões de Pessoas Dispensáveis**

---

## Sobre o Livro

Este livro explora uma das questões mais urgentes do nosso tempo: o que acontecerá com bilhões de trabalhadores quando a inteligência artificial puder fazer seus trabalhos melhor, mais rápido e mais barato?

Não é ficção científica. É uma análise baseada em dados, pesquisas e projeções de especialistas sobre um futuro que já começou.

## Gerando o PDF

### Pré-requisitos

- [Pandoc](https://pandoc.org/installing.html) instalado
- [XeLaTeX](https://www.latex-project.org/get/) instalado (via MacTeX ou TeX Live)

### Comando

```bash
chmod +x gerar-pdf.sh
./gerar-pdf.sh
```

O PDF será gerado como `a-classe-inutil.pdf`.
